## `data.json` 文件说明

`data.json` 文件包含了以上表格的所有数据，方便开发者通过程序访问和使用这些数据。以下是 `data.json` 文件的数据结构说明：

```json
[
  {
    "id": 1,
    "Icon": "https://xx.xx.xx/1/1fbbd020ff3e1cd8175ac5977712f9b3.webp",
    "Name": "AItianhu",
    "Description_CN": "无限制的GPT-3.5 服务",
    "Description_EN": "Unlimited GPT-3.5 service",
    "Tags": "GPT-3.5, 免费, 无限制",
    "Link": "https://chat10.aiyunos.top",
    "Status": "✅"
  },
  // ... 其他站点数据
]
```

*   `id`: 站点的唯一数字 ID，保持连续，请勿修改或重复使用其他网站的 `id`。
*   `Icon`: 站点图标的 URL。
*   `Name`: 镜像站点的名称。
*   `Description_CN`: 站点简要中文描述。
*   `Description_EN`: 站点简要英文描述。
*   `Tags`: 站点的标签，用于描述站点特性，方便搜索，使用英文逗号 `,` 分隔。
*   `Link`: 镜像站点的 URL。
*   `Status`: 站点可用状态：`✅` 可用；`🚧` 部分可用或维护中；`🔴` 不可用。
